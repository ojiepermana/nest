# @ojiepermana/nest

NestJS Core Library - Core utilities and common modules for NestJS applications.

## Installation

```bash
npm install @ojiepermana/nest
```

## Usage

```typescript
import { NestModule } from '@ojiepermana/nest';

@Module({
  imports: [NestModule],
})
export class AppModule {}
```

## Features

- Common utilities
- Shared modules
- Helper functions
